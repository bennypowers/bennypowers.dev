Welcome to the Desktop Demo

This is a full integration demo of the GNOME 2 desktop shell. It includes the top panel with session menus and clock, desktop icons, and a managed window.

Test scenarios:

  • Click the Applications/Places/System menus
  • Click the window titlebar buttons (minimize, maximize, close)
  • Drag the window by its titlebar
  • Double-click the titlebar to maximize/restore
  • Click desktop icons

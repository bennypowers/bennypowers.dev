Custom Elements

This site uses custom HTML elements to recreate the GTK2 widget toolkit:

  • <gtk2-window> - window frame with titlebar and controls
  • <gtk2-menu-button> - panel menu trigger with dropdown
  • <gtk2-menu-item> - menu item with icon, label, and submenu support
  • <gtk2-scrolled-window> - scrollable content area
  • <gnome2-panel> - top and bottom panels
  • <gnome2-desktop> - desktop container with icon grid and window area
  • <desktop-icon> - desktop shortcut icon

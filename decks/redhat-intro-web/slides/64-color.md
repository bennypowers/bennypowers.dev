Colours in CSS can be represented multiple ways.


Hex Values
: <code>#<span style="color:red">FF</span><span

RGB function
: ```css
  rgb(255, 255, 255, 1);
  ```

HSL function
: ```css
  hsl(360deg 100% 100% / 100%);
  ```

[Named Colours][named]

  ```

Others
